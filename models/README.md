# Required Model Assets

This directory must contain the following trained assets for the MVP to function:

1.  **`best_model.pth`**: The trained PyTorch LSTM model checkpoint.
2.  **`scaler.joblib`**: The fitted `StandardScaler` object used during training.
3.  **`yolov8n-pose.pt`**: The YOLOv8 pose estimation model. You may need to download this if it's not already present.
